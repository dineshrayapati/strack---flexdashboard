# strack---flexdashboard
Flex dashboard demonstrating data visualization in R for Strack R GESOC Project.
## How to Run the Flexdshboard
1. Install R and RStudio.
2. Instal required packages like 'flexdashbord', 'ggplots2'.
3. Open the '.Rmd' fileinRStudio.
4. Click **Run Document** to render the dashboard.
   This project demostrates a simple data visualization dashboard inR for the Strack R GSoC preparation.
